# 快速开始指南

## 🚀 一键启动

```bash
npm run dev
```

就这么简单！这个命令会自动启动：
- ✅ 后端服务器（Express）
- ✅ 前端界面（React + Vite）
- ✅ 数据库连接（MySQL）
- ✅ MQTT 数据接收

## 📋 前置要求

1. **Node.js**（已安装 ✅）
2. **MySQL**（需要启动）
   - 数据库：jiankong_zhongdei
   - 用户：root
   - 密码：123456

## 🎨 界面功能

系统包含 5 个主要界面：

1. **主画面** - 反应槽温度、流量数据
2. **罐区** - 储罐液位、泄漏检测
3. **装车** - 装车流量监控
4. **外部设备** - 聚铝老厂和滚筒干燥设备
5. **车间选择** - 车间切换

### 报警系统

- 顶部显示报警数量
- 右侧报警面板（有报警时自动显示）
- 按类型分类：温度、液位、泄漏、电流、流量

## 🔧 常见问题

### Q1: 端口被占用？
**A**: 修改 `server.ts` 中的 `PORT = 3000` 为其他端口

### Q2: 数据库连接失败？
**A**: 
1. 检查 MySQL 是否启动
2. 确认密码：`.env.local` 中的 `DB_PASSWORD=123456`
3. 确认数据库存在或有创建权限

### Q3: 没有接收到 MQTT 数据？
**A**: 
1. 等待约 5 秒（MQTT 推送间隔）
2. 检查网络连接
3. 查看控制台是否有错误信息

### Q4: 前端页面打不开？
**A**: 
1. 确认服务器已启动
2. 检查浏览器地址：http://localhost:3000
3. 清除浏览器缓存
